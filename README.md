# 🚀 Atividades de Lógica com TypeScript

Este repositório contém uma coleção de exercícios práticos desenvolvidos para exercitar conceitos de lógica de programação, tipagem e funções utilizando **TypeScript**.

---

## 🛠️ Tecnologias Utilizadas

* **TypeScript**: Linguagem principal para o desenvolvimento das funções.
* **Node.js**: Ambiente de execução para rodar os scripts.

---

## 📂 Atividades Desenvolvidas

Abaixo estão detalhadas as três atividades presentes neste repositório:

### 1. Cálculo de Desconto (`atividadeUm.ts`)
Uma função que recebe o valor original de um produto e a porcentagem de desconto, retornando o valor final calculado.
* **Conceitos:** Operações aritméticas e interpolação de strings.

### 2. Teorema de Pitágoras (`atividadeDois.ts`)
Implementação para encontrar o valor da hipotenusa a partir de dois catetos fornecidos.
* **Fórmula:** $$a^2 + b^2 = c^2$$
* **Conceitos:** Uso da biblioteca `Math` (`sqrt` e `pow`).

### 3. Saudação Inteligente (`atividadeTres.ts`)
Uma função que retorna uma saudação personalizada ("Bom dia", "Boa tarde" ou "Boa noite") baseada no nome do usuário e na hora informada.
* **Conceitos:** Estruturas condicionais aninhadas e validação de dados.
